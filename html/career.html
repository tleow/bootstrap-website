What is Information Technology?

Information Technology (IT) is the driving force behind many digital transformations we see and use today, including the Smart Nation vision.

In Singapore, there continues to be a strong demand for IT professionals in Software Development and User Experience (UX) design, and in Immersive Simulation Technologies such as Augmented and Virtual Reality.

Career Options

Web Developer

Web developers design the appearance, navigation and content organization of a website. They use coding languages such as HTML, CSS and JavaScript.

Job requirements:

1. Bachelor's degree in Web development or related field, or relevant experience.

2. Solid knowledge and experience in programming applications.

3. Proficient in JavaScript, HTML, CSS. Proficient in My SQL.

Software Developer

A Software Developer is a professional who is charged with designing and coding software for businesses and consumers alike. They use programming languages like Java or C++ to create programs.

Job requirements:

1. A bachelor's degree in software engineering, computer science, or a similar field.

2. The desire to work in a fast-paced environment.

3. Deep programming language knowledge.

4. Experience working on a variety of software development projects.

Software Engineer

Software engineers apply their knowledge of mathematics and computer science to create and improve new software.

Job requirements:

1. A bachelor's degree in computer science, software engineering, or another related field.

2. 5 to 7 years of software engineering or software development experience, preferably in a related field.

3. Excellent coding skills.

4. A comprehensive understanding of computer science concepts.

5. Hands-on programming experience using relevant languages.

Data Scientist

A data scientist analyzes and organizes data to determine trends that can influence business decisions.

Job requirements:

1. Find patterns and trends in datasets to uncover insights.

2. Create algorithms and data models to forecast outcomes.

3. Deploy data tools such as Python, R, SAS, or SQL in data analysis.

4. Use machine learning techniques to improve the quality of data or product offerings.
